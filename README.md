# 模卡选角系统

一个本地可运行的模特资料库与客户需求匹配工具。当前版本是纯前端静态应用，数据会保存在浏览器本地存储里，适合先跑业务流程和整理字段。

## 启动

```bash
python3 -m http.server 5173
```

打开：

```text
http://localhost:5173
```

客户链接默认只显示需求提交页：

```text
http://localhost:5173/
```

后台管理入口用于导入资料和查看资料库：

```text
http://localhost:5173/?admin=1
```

## 已支持

- 客户默认只看到需求提交页面，提交后才看到推荐清单
- 后台入口包含四个页面：客户需求提交页面、模特资料导入页面、模特资料库、小红书推广
- 录入和编辑模特资料，支持 CSV 批量导入
- 按性别、年龄、身高、城市、预算、风格标签和额外要求自动评分
- 匹配结果按匹配度、报价、身高、更新时间排序
- 推荐清单一键复制，方便发给客户
- 推荐清单底部可一键复制经纪人联系方式
- CSV 导入和导出资料库
- 每周根据模特资料库生成 3 篇小红书推广帖，支持单模特推荐贴和类型模特合集贴
- 客户需求保存到本地，下次打开可恢复

## CSV 字段

导入 CSV 支持英文或中文表头：

```csv
name,gender,age,height,measurements,city,fee,availability,tags,portfolio,notes
林安娜,女,23,172,82/60/88,上海,2200,6月可约,"高级感|画册|美妆",https://example.com,镜头表现稳定
```

中文表头也可以使用：`姓名`、`性别`、`年龄`、`身高`、`三围`、`体型`、`城市`、`报价`、`档期`、`标签`、`风格标签`、`模卡链接`、`备注`。
